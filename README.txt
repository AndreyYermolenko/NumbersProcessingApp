Программа позволяет считать массив целых чисел из текстового файла и обработать их.
Числа в файле должны быть разделены между собой с помощью "\n" (каждое число на новой
строке).

Результат работы программы состоит из данных:
    1. максимальное число в файле;
    2. минимальное число в файле;
    3. медиана;
    4. среднее арифметическое значение;
    5. наибольшая последовательность идущих подряд чисел, которая увеличивается;
    6. наибольшая последовательность идущих подряд чисел, которая уменьшается;

Реализация программы предполагает, что может быть несколько возрастающих или убывающих
последовательностей с максимальной длинной. В таком случае будут выведены все последова-
тельности с максимальной одинаковой длинной. Например, если файл состоит из следующих
данных:
1
2
1
2
3
1
2
3

Будут выведены:
* две возрастающие последовательности:
    1, 2, 3;
    1, 2, 3;
* две убывающие последовательности:
    2, 1;
    3, 1
